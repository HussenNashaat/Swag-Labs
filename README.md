# SwagLabs Test Automation Project

This project showcases automated testing for the Swag Labs web application using Selenium WebDriver and TestNG.  
It follows the Page Object Model (POM) design pattern to keep the code clean, modular, and maintainable.

---

## 🛠 Tools Used

- Java
- Selenium WebDriver
- TestNG
- Maven

---

## 📁 Project Structure

SwagLabs/
├── src/
│ ├── main/java/pages/ → Page Object classes (LoginPage, HomePage, etc.)
│ └── test/java/tests/ → Test classes (LoginTest, etc.)
├── pom.xml → Maven dependencies
├── testng.xml → TestNG test suite
└── README.md

---

## ▶️ How to Run the Tests

1. Clone the repository: git clone https://github.com/HussenNashaat/Swag-Labs.git

2. Open the project in IntelliJ IDEA or VS Code.

3. Run the tests using Maven: mvn test

---

## ✅ Test Scenarios Included

- Login with valid and invalid credentials
- Add to cart and remove from cart
- Checkout process
- Logout and session management
